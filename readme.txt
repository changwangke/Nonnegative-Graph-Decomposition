Code for "Nonnegative Graph Decomposition" in MATLAB
MATLAB version: R2023b. (Requires Statistics and Machine Learning Toolbox and Deep Learning Toolbox installed.)
Open the folder "NGD" and run "TestSythDataset1byNGD.m" to see a demonstration case.
To test other datasets, change the dataset name in the load_dataset('XXXXX') function.
Other folders contain comparison methods and additional datasets used in paper.
